# 🛒 RetailMart Analytics Dashboard

End-to-end analytics project built using PostgreSQL, SQL pipelines, and a web dashboard.

## 🚀 Features

* Sales, Customer, Product, Finance analytics
* Automated refresh using shell scripts
* JSON-based dashboard data
* Modular SQL pipeline

## 🛠 Tech Stack

* PostgreSQL
* SQL
* Bash
* HTML / CSS / JavaScript

## ⚙️ How to Run

1. Run SQL files (01_setup → 03_kpi_queries → 04_alerts)
2. Run:
   bash 05_refresh/export_all_json.sh
3. Open:
   06_dashboard/index.html
